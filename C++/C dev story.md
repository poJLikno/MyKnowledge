C89 / C90 (ANSI C / ISO C)
Первая официальная стандартизация языка.
 * Прототипы функций: Появилась возможность описывать аргументы функций (раньше была только проверка имени).
 * Новые типы: Добавлены void, const, volatile, enum, а также signed.
 * Стандартная библиотека: Впервые официально закреплены stdio.h, stdlib.h, string.h и другие.
 * Локаль: Поддержка национальных наборов символов.
C95
Минорное обновление для поддержки интернационализации.
 * Wide Characters: Добавлены библиотеки <wchar.h> и <wctype.h> для работы с «широкими» символами.
 * Диграфы и триграфы: Альтернативные комбинации символов для клавиатур, где нет {, }, [ и т.д.
C99
Самое масштабное обновление, сделавшее язык гораздо удобнее.
 * Типы данных: Появились long long, _Bool, complex.
 * stdint.h: Введены типы с фиксированной шириной (int32_t, uint64_t).
 * Комментарии: Официально разрешены однострочные комментарии //.
 * VLA (Variable Length Arrays): Массивы переменной длины (размер определяется в рантайме).
 * Inline-функции: Возможность советовать компилятору встраивать код функции.
 * Designated Initializers: Инициализация структур по именам полей: .x = 10.
 * Объявление переменных: Разрешено объявлять переменные в любом месте кода (не только в начале блока) и внутри цикла for.
C11
Фокус на многопоточности и безопасности.
 * Многопоточность: Добавлены <threads.h> и атомарные операции <stdatomic.h>.
 * _Generic: Поддержка «генериков» (выбор кода в зависимости от типа аргумента).
 * Static assertions: Проверки на этапе компиляции через _Static_assert.
 * Безопасность: Новые функции с проверкой границ (например, gets_s вместо опасного gets).
 * Анонимные структуры и объединения: Упрощают вложенность данных.
C17 (C18)
Техническое обновление.
 * Баг-фиксы: Не принес новых функций. Исправлены ошибки в спецификации C11 и уточнены спорные моменты. Это текущий индустриальный стандарт «по умолчанию».
C23
Современный облик языка (финализирован, внедряется в компиляторы).
 * nullptr: Наконец-то введено полноценное ключевое слово для нулевого указателя.
 * constexpr: Возможность определять константы, вычисляемые при компиляции.
 * bool, true, false: Стали полноценными ключевыми словами (раньше требовали <stdbool.h>).
 * auto: Вывод типа переменной (как в C++).
 * #embed: Прямая вставка бинарных данных в исходный код.
 * Атрибуты: Введен синтаксис [[deprecated]], [[maybe_unused]] и др.
 * Удаление K&R функций: Старый стиль объявления функций (без типов аргументов) больше не поддерживается.
